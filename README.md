# NovaDash Admin Panel

NovaDash is a premium, responsive admin dashboard template built with HTML, CSS, and JavaScript. It features a modern design, dark/light mode support, and interactive components.

## Features

-   **Responsive Layout**: Fully responsive sidebar and grid system.
-   **Dark/Light Mode**: Built-in theme switcher with local storage persistence.
-   **Interactive Charts**: Integrated Chart.js for beautiful data visualization.
-   **Components**: Stats cards, data tables, forms, and more.
-   **Clean Code**: No frameworks, just pure HTML/CSS/JS.

## Pages Included

1.  **Dashboard** (`dashboard.html`): Main overview with stats and charts.
2.  **Users** (`users.html`): User management table.
3.  **Analytics** (`analytics.html`): Detailed charts and reports.
4.  **Settings** (`settings.html`): User profile and application settings.

## Installation

1.  Clone the repository or download the files.
2.  Open `dashboard.html` in your browser.
3.  No build step required!

## Customization

-   **Colors**: Edit the CSS variables in `css/style.css` to change the theme colors.
-   **Charts**: Modify `js/main.js` to update chart data and options.

## Technologies Used

-   HTML5
-   CSS3 (Variables, Flexbox, Grid)
-   JavaScript (ES6+)
-   [Chart.js](https://www.chartjs.org/)
-   [Boxicons](https://boxicons.com/)
-   [Google Fonts (Inter)](https://fonts.google.com/)
